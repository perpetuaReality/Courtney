Command list

and common parameters

If a command requires a parameter (also called variables or arguments), it will most likely be one of these:
[CHARACTER] The name of a character. These have to be shorter than 32 characters and are case-insensitive. If you want to use spaces, surround this parameter by single quotes. (Ex. 'Lengthily Named Character')
[MAX HP] The maximum HP of a character. These have to be positive integers smaller than 50 million.
[NEW HP] The new current HP of a character, if you want to edit it directly. Can be positive or negative (or zero), but has to be between positive and negative 50 million.
[CHANGE]
[DAMAGE]
[HEALING]		 A change to the HP of a character. These can be positive or negative (or zero), but have to be between positive and negative 1 million.
